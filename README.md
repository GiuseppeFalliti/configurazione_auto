# Gestione Auto - Configuratore

Un semplice configuratore di automobili sviluppato in **HTML**, **CSS** e **JavaScript**. Gli utenti possono personalizzare un'auto selezionando il tipo di motore, versione, rivestimento interno, assistenza alla guida e colore. Il costo totale viene calcolato dinamicamente in base alle scelte effettuate.

---

## Funzionalit√†

- **Selezione del motore**: Benzina, Diesel, Ibrido o Elettrico.
- **Scelta della versione**: Base, Sportiva o SUV.
- **Personalizzazione degli interni**: Tipo di rivestimento (Tessuto o Pelle) e colore.
- **Assistenza alla guida**: Base o Avanzato.
- **Calcolo dinamico del prezzo totale**: Il costo totale viene aggiornato automaticamente in base alle opzioni selezionate.

---

## Struttura del Progetto

### File principali:
1. **`index.html`**: Contiene la struttura HTML dell'applicazione.
2. **`style.css`**: Gestisce lo stile visivo dell'interfaccia.
3. **JavaScript (incluso inline)**: Gestisce la logica del calcolo del prezzo e l'aggiornamento dinamico delle opzioni.

---

## Esempio di utilizzo

1. Clona o scarica il progetto sul tuo computer:
   ```bash
   git clone https://github.com/tuo-username/gestione-auto.git
   ```
2. Apri il file `index.html` in un browser moderno.
3. Personalizza l'auto selezionando le opzioni preferite.
4. Visualizza il prezzo totale aggiornato in tempo reale.
